cli(comand line interface)

ls (-l,-al 등 사용가능) : 현재 디렉토리 파일 목록 출력
ls -a(--all) : 모든파일 출력 (감춰진 파일은 앞에 '.'이 붙는다.)
pwd : 현재 위치하고 있는 디렉토리 알려주기

mkdir 디렉토리명 : 새로 디렉토리 생성
mkdir -p(--parents) 디렉토리/디렉토리/디렉토리 : 필요하면 부모디렉토리까지 생성

cd 디렉토리면: 디렉토리 이동
cd .. :  부모 디렉토리로 이동 
cd / : 최상의 디렉토리로 이동

rm 파일명 : 파일 삭제
rm -r 디렉토리명

명령어 --help : 해당 명령어의 사용설명서
man 명령어 : 해당 명령어의 자세한 사용설명서
q : 빠져나가기
/문자열 : 찾고자 하는 문자열
n : 다음문자열 찾기

sudo(super user do)

기타 인터넷으로 찾아보기
